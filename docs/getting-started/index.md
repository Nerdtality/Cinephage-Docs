# Getting started

Welcome to Cinephage. This section contains tutorials to help you get up and running.

## What You Will Learn

These tutorials guide you through:

1. Installing Cinephage with Docker
2. Configuring essential settings
3. Adding your first media
4. Understanding the interface

## Prerequisites

Before starting, you will need:

- A server or computer to run Cinephage
- Docker and Docker Compose installed
- Basic familiarity with command line tools
- A TMDB API key (free)

## Tutorial Sequence

Follow these tutorials in order:

### 1. Installation

Learn how to install Cinephage using Docker Compose. This covers:

- Docker Compose configuration
- Environment variables
- Volume mounts
- First run verification

[Start with Installation →](installation)

### 2. Initial Setup

Configure Cinephage for first use:

- TMDB API key setup
- Root folder configuration
- Download client setup
- Basic settings

[Continue to Initial Setup →](initial-setup)

### 3. Adding Media

Add your first movie or TV series:

- Searching TMDB
- Adding to library
- Understanding monitoring
- Running your first search

[Learn to Add Media →](adding-media)

### 4. Understanding the Interface

Navigate the Cinephage UI:

- Dashboard overview
- Library management
- Activity monitoring
- Settings organization

[Explore the Interface →](understanding-interface)
